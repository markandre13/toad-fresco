FieldSubject [index]

declared in file: /opt/Fresco/Fresco/defs/widgets.idl

A field subject is a string of text.
long count();
Return the number of characters in the string.
void append_char(in CharCode c);
void prepend_char(in CharCode c);
void insert_char(in long index, in CharCode c);
void insert_chars(in long index, in CharString str);
Inserting at index 0 prepends and inserting at index count()-1 appends to the string.
void replace_char(in long index, in CharCode c);
void replace_chars(in long first, in long last, in CharString str);
void replace_all(in CharString str);
void delete_chars(in long first, in long last);
void delete_all();
These operations are used to delete or replace regions of the string.
CharCode get_char(in long index);
string get_chars(in long first, in long last);
string get_all();
Returns copy of char, string, or string within given region.
void copy(in FieldSubject fs);
Copy the given field subject into this subject.
