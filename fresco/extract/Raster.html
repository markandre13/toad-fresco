Raster [index]

declared in file: /opt/Fresco/Fresco/defs/drawing.idl
ancestors: FrescoObject

A raster is a two-dimensional array of primitive elements, which include a color and a blending factor between zero and one. Zero means nothing is blended and one means everything is blended. . There are three uses of rasters: (1) as a color image, (2) as a monochrome image, where the colors are only black or white, and (3) as a mask, where the blending factors are only zero or one. In X terminology, uses (2) and (3) are called a "bitmap," while use (1) is a "pixmap."
typedef long Index;
The Index type defines the range of values for indices into the array of elements.
struct Info {
  Index rows;
  Index columns;
  Index origin_x;
  Index origin_y;
  
};
The Info structure defines the dimensions, in terms of rows and columns, and the origin for the raster.
boolean equal(in Raster r);
unsigned long hash();
The equal operation tests whether two raster objects refer to the same image or bitmap data. The hash operation will return the same value for two equal rasters, though there is also some chance it will return the same value for two unequal rasters.
void raster_info(out Info i);
Return the metric information for this raster.
void set_color(in Index row, in Index col, in Color c);
void set_color_blend(in Index row, in Index col, in Color c, in float blend);
void set_mask(in Index row, in Index col, in boolean on);
These operations write the raster element specified by a row and column. The operations have no effect if either the row or column are out of range for the raster. . The set_color operation indicates a color and assumes a blending factor of one. The set_color_blend operation specifies both the color and the blending factor. The set_mask operation does not specify a color and indicates a blending factor of one (if the boolean parameter is true) or zero (if the parameter is false).
Color get_color(in Index row, in Index col);
void get_color_blend(in Index row, in Index col, out Color c, out float blend);
boolean get_mask(in Index row, in Index col);
These operations read the raster element specified by a row and column. The operations have no effect if either the row or column are out of range for the raster. . The get_color operation returns the color, get_color_blend returns both the color and the blending factor, and get_mask returns true if the blending factor is non-zero.
Coord scale;
Define the mapping between raster indices and coordinates. If the scale is zero (the default value), then one index will correspond to one pixel on the output device. Otherwise, the scale defines the size of a raster element in resolution-independent units.
readonly attribute unsigned long version;
Each time the raster data is changed the version number changes so that objects that cache raster data can tell when the cached data is stale.
