TextFlow::Observer [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl

interface for a Text::Subject observer. An observer can register to receive notifications of change for the portion of a subject that it is interested in.
void insert(in Range rng);
notification that there was an insert into the document. The given range bounds the freshly inserted region.
void remove(in Range rng);
notification that a portion of the document has been removed. The range is given in terms of what the view last saw (ie before updating sticky positions).
void changed(in Range rng);
An attribute changed for the given range.
void element_insert(in Element elem, in Element right_sib);
void element_remove(in Element elem);
notification of structural changes. These are sent to the parent element. The given element is the element that was inserted or removed.
