TextFlow::View [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl
ancestors: Glyph, Observer

A view that represents some document subject and provides a mapping between the two.
Subject get_subject();
Range get_range();
Element element();
get_subject fetchs the document subject associated with this view. get_range retrieves the amount of coverage within the subject that this view applies. get_element fetchs the element associated with this glyph.
boolean position_to_coord(in Position pos, inout Vertex lower, inout Vertex upper);
Position coord_to_position(in GlyphTraversal t, in Vertex pt);
position_to_coord and coord_to_position provide a mapping between subject and view. The coordinates are in terms of the window the glyph is contained in.
void damage_range(in Range rng);
cause a region corresponding to a portion of the subject to be redrawn.
boolean can_insert(in Position pos);
indicates whether the given position can be used as an insert position within this view.
Breakable break_behavior();
return the breakability of the view, if the glyph supports breaking by a formatter.
void background(in Glyph bg);
set background of the view... if the view supports backgrounds.
