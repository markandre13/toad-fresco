AutoScroll [index]

declared in file: /opt/Fresco/Fresco/defs/layouts.idl

An auto-scroll assists a viewer in automatically scrolling the view when the user is interacting with something in the view and the mouse is moved past the edge of the viewers allocation.
struct PreferencesInfo {
  float initial_delay;
  float repeat_delay;
  
};
The initial delay controls how much delay there is before the first auto-scroll occurs. The repeat delay controls how much delay there is between successive auto-scrolls.
AutoScroll::PreferencesInfo preferences;
The initial delay controls how much delay there is before the first auto-scroll occurs. The repeat delay controls how much delay there is between successive auto-scrolls.
boolean handle(in GlyphTraversal traversal, in Event e);
Handle update and stop based on event location relative to the allocation. Returns true if the auto-scroller is active.
void auto_scroll(in float x_amount, in float y_amount);
Auto-scroll using the scroll amounts. The scroll amounts are fractions of the adjustments small scroll. So to scroll by the small scroll amount to the upper x and lower y use an x amount of +1 and a y amount of -1.
void stop();
Stop auto-scrolling.
