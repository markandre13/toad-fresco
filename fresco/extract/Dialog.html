Dialog [index]

declared in file: /opt/Fresco/Fresco/defs/dialogs.idl
ancestors: FrescoObject

A Dialog is a window that can be posted in a non-modal or modal fashion.
void show(in Window show_for);
void hide();
These operations control the visibility of the dialog in a non-modal fashion.
void done();
This operation informs a modal dialog that it should return after processing of the current event is completed.
void post_modal(in Window show_for);
This operation posts the dialog in a modal fashion, grabs input, dispatches events until the until the user responds, then returns.
