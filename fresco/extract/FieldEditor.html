FieldEditor [index]

declared in file: /opt/Fresco/Fresco/defs/widgets.idl
ancestors: Viewer

A field editor is a viewer that allows editing a single line of text.
ExclusiveEditMgr field_edit_manager();
FieldSubject field_subject();
string get_current();
A field editor may be attached to a field subject. It changes this subject (and notifies its observers) whenever text modifications are "committed" using the return key. If the subject is independently updated, then, upon update notification, the field editor changes its current contents to match the subject. The get_current operation provides direct access to the field editor's current contents. Note that this may not necessarily match the current value of the field subject.
FrescoObject enter_notifier();
FrescoObject input_notifier();
The enter_notifier operation returns an object which notifies its observers whenever modifications are "committed" using the return key. The input_notifier operation returns an object which notifies its observers when each visible character is input into the field editor. (Visible characters do not include such things cursor movement and return keys.)
Adjustment scroll_adjustment();
This operation returns an adjustment that changes whenever scrolling occurs. A scrollbar, for example, can be attached to this adjustment.
