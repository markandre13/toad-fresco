Viewport [index]

declared in file: /opt/Fresco/Fresco/defs/layouts.idl
ancestors: Viewer

A viewport allows adjustment of the view of a glyph.
enum SpanAllocationMode {
   natural, given
};
The natural mode means that the viewport allocates the natural span requested by its child. The given mode means the viewport allocates its given span.
void set_allocation_mode(in SpanAllocationMode mode, in Axis a);
SpanAllocationMode get_allocation_mode(in Axis a);
By default, a viewport allocates the natural span requested by its child in both the X and Y dimensions. The set_allocation_mode operation can be used to change this behavior for a given dimension. The get_allocation_mode returns the current mode for the given dimension.
