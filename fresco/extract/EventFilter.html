EventFilter [index]

declared in file: /opt/Fresco/Fresco/defs/event.idl
ancestors: FrescoObject

An event filter controls the routing of events received by a display.
boolean handle(in Event e);
The filter should return true if it does not want the event to be processed by any other filters or the event's target window.
