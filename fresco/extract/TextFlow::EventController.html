TextFlow::EventController [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl

This interface forms a simple hook for mapping events to edits, navigation, etc of a document editor. One might choose to map events to actions in a different language from that used for the implementation of a document editor itself.
boolean press(in GlyphTraversal t, in Event e);
boolean drag(in GlyphTraversal t, in Event e);
boolean release(in GlyphTraversal t, in Event e);
boolean key_press(in GlyphTraversal t, in Event e);
void focus_in();
void focus_out();
