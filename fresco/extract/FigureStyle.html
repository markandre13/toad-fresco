FigureStyle [index]

declared in file: /opt/Fresco/Fresco/defs/figures.idl
ancestors: StyleContext

A figure style provides a type-checked interface to the predefined figure attributes.
Color background;
Figures with a "fill_stroke" mode use the background color for filling.
Brush brush_attr;
Figures with a "stroke" or "fill_stroke" mode use the brush to draw the outline of the figure.
Font font_attr;
Labels use the font attribute for drawing and computing the bounding box of labels.
Color foreground;
Figures with a "fill" mode use the foreground color for filling. Figures with a "stroke" or "fill_stroke" mode use the foreground color to draw the outline of the figure.
