TextFlow::Element [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl
ancestors: FrescoObject

Structural element of the document. A completely unstructured document will simply have one element. An element is meant to be somewhat like an SGML element.
Tag register_interest(in Observer view);
void unregister_interest(in Tag t);
register_interest allows an observer to arrange to be notified of changes that occur within its range of interest. unregister_interest allows the observer to discontinue receiving notifications.
