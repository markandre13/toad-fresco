DisplayContext [index]

declared in file: /opt/Fresco/Fresco/defs/display.idl

A display context allows one to make connections to displays (window servers).
Display open_string(in string n);
Display open_default();
Return a reference to a specified display or the default display. The name of the default display is platform-dependent. On POSIX systems, the default display name is the value of the DISPLAY environment variable. . Display strings are of the form N:D.S, where N is an optional network host name. If omitted, then the display is assumed to be on the local host. D is a display number on the host, and S is the number of the screen to use on the display by default.
