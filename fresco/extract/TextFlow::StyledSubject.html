TextFlow::StyledSubject [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl

interface for a styled document. The StyledSubject implements a simple set of policies that maintain various element structures over the document that resemble the Rich Text Format (RTF) Specification .
