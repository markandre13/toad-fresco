TextFlow::ViewFactory [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl

A glyph factory that can be used to build pieces of a documents view. The factory might build new glyphs for each request, or return shared glyphs.
View build(in Element elem, in Editor parent);
return a glyph to be used to represent a particular document element. The factory might build a new glyph or might return a shared glyph. In the process of building an element, a builder might call other factories if the element has children. The factory should register the offset of the glyph with the parents mapped view. If the created glyph is a viewer, it should be connected to the parents in the focus hierarchy if appropriate.
CharString build_name();
name of the factory. This is used to create a binding through a naming context from element types.
