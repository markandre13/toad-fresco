TextFlow::Composition [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl
ancestors: Glyph

A Composition tiles its children along an axis, creating breaks as needed when no more children will fit. Breaks will only be created in glyph locations (ie offsets) that have the Breakable interface registered at it. One can use the normal glyph interface and register breakable interfaces with the offsets, or use the alternative interface that takes a breakable when adding glyphs.
void breakable_append(in Glyph g, in Breakable b);
void breakable_prepend(in Glyph g, in Breakable b);
long num_lines();
CompositionIter get_iterator(in long line);
The function get_iterator gives an iterator positioned over the given line. The lines are indexed starting at 0. The num_lines function gives the total number of lines contained.
Glyph background;
The background is rendered after formatting has been updated but before the children of the composition.
