DialogKit [index]

declared in file: /opt/Fresco/Fresco/defs/dialogs.idl
ancestors: FrescoObject

DialogKit provides operations for creating common types of message dialogs.
MessageDialog error(in CharString title, in Glyph message);
Create an error message dialog with ok, cancel, and help buttons.
MessageDialog information(in CharString title, in Glyph message);
Create an information message dialog with ok, and help buttons.
MessageDialog question(in CharString title, in Glyph message);
Create a question message dialog with yes, no, cancel, and help buttons.
MessageDialog warning(in CharString title, in Glyph message);
Create a warning message dialog with ok, cancel, and help buttons.
MessageDialog working(in CharString title, in Glyph message);
Create a working message dialog with a stop button.
ChooserDialog file_chooser(in string initial_directory);
ChooserDialog name_chooser(in CosNaming::NamingContext initial_context);
Return a chooser for files or names, given an initial place from which to start.
PromptDialog prompt(in CharString title, in Glyph message, in FieldSubject subj);
Create a prompt dialog with ok, apply, reset, cancel, and help buttons. If subject argument is nil, one will be created. Return the default glyph for the various flavors and choices.
