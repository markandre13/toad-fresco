StyleVisitor [index]

declared in file: /opt/Fresco/Fresco/defs/style.idl
ancestors: FrescoObject

A style visitor is essentially a callback for iterating over the elements of a style context. The visitor can be used to access the aliases, attributes, and style children of a particular context.
boolean visit_alias(in CharString n);
boolean visit_attribute(in StyleValue a);
boolean visit_style(in StyleContext s);
Each of the visit operations returns a boolean value indicating whether the iteration should continue or not. This module contains definitions for text support.
