MenuItem [index]

declared in file: /opt/Fresco/Fresco/defs/menu.idl

MenuItems are used to define the content of menus.
typedef sequence Path;
Paths are used to identify how to reach one menu item from another.
string key;
Each menu item in a menu should have a unique key so that a specific menu item can be searched for.
MenuItem parent;
Child menu items point back to their parents. The parent attribute can be used to get or set a childs parent.
Tag attach(in MenuItemObserver observer);
void detach(in Tag attach_tag);
Menu items can be observed for changes by attaching a menu item observer to the menu item using the attach member. Menu item observers are not reference counted by menu items because we assume that the menu item observer will detach before its life is over.
enum TypeId {
   submenu, leaf, separator
};
The types represent general types of menu items. Submenu items contain more menu items. Leaf items are things like buttons, check boxes, etc. A separator item is used to visually indicate separate groups of items. Get the items type.
Glyph label_glyph;
attribute string label_string;
attribute string font_family;
// nil means defaultattribute string font_style;
// nil means defaultattribute Coord font_size;
// 0 means default
These attributes are used to create the visual representation of the menu items label. At least one of the label glyph or string must be provided.
enum MarkId {
   unmarked, checkmark, radiomark
};
The marks represent different types of leaf menu items. Checkmarks are used for on/off states and radiomarks are used for one of many items. The mark determines the visual representation of the mark that appears in the menu item.
Telltale mark_state;
The mark state controls the appearance of a menu items mark.
boolean underline;
The underline attribute is used to underline the label in a menu item.
boolean more_dialog;
The more dialog attribute is used to indicate that if the menu item is chosen the user will be prompted for more information before the action completes.
KeySym mnemonic;
The mnemonic is used to traverse open menus using the keyboard.
KeyChord accelerator;
The accelerator is used to execute an menu item action using the keyboard.
boolean enabled;
The enabled state of the menu item is used to enable/disable execution of the menu items action.
Action item_action;
The item action attribute is used to execute the menu items functionality.
readonly attribute long count;
The count is the number of child menu items within a composite menu item.
MenuItem item(in long i);
The item function can be used to access a specific child menu item.
void prepend(in MenuItem item);
void append(in MenuItem item);
void insert(in long i, in MenuItem item);
void remove(in long i);
void remove_all();
Child menu items are added/removed using the prepend, append, insert, remove, and remove_all functions.
MenuItem default_item;
The default_item function returns the child menu item that should be the default. If there is no default then nil will be returned.
void notify_observers(in MenuItemObserver::Aspect a);
Send to all observers the given aspect-notification
