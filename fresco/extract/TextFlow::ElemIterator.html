Element current_child();
The current_child operation returns the child element pointed to. If no child is pointed to or there are no children, nil is returned. The insertion functions insert an element and leaf the iterator positioned at the inserted element (ie to fetch the freshly inserted element one would call current_child()).
void visit_parent();
void visit_child();
visit_parent positions the iterator to its parent. visit_child positions the iterator at the child it currently points to.
