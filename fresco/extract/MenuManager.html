MenuManager [index]

declared in file: /opt/Fresco/Fresco/defs/menu.idl
ancestors: FrescoObject

A menu manager is the controller for a menu bar. It examines the events destined for the a menu bars window and activates the menu bar when appropriate.
MenuItem root();
The root function returns the root menu item that was used to construct the menu bar.
void root_menu(in Menu m);
The root menu to be used when opening menus should be set before opening the menus or handling viewer events. The menu should have been created using the root menu item.
boolean handle_viewer_event(in GlyphTraversal t, in Event e);
The handle viewer event function deals with key events that open/close menus and with accelerators.
void open_menus(in Window w);
void close_menus();
The open menus and close menus functions activate and deactivate menu bar event processing. The open menus function should be passed the window that contains the menu bar viewer.
Screen filter_screen();
The filter_screen function returns the screen of the window that is watched for menu bar events.
Tag add_window(in Window w);
void remove_window(in Tag t);
These functions allow events to be passed to the indicated window during menu bar event processing. They are typically used by pulldown menus created from the menu bar.
void select(in MenuItem item, in boolean flap);
MenuItem selection();
void unselect();
The selection functions are used to change or query the current selection. The flap flag indicates that submenus should be displayed for the selected item.
void activate();
The activate function activates the currently selected menu item.
