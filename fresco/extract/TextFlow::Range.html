TextFlow::Range [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl
ancestors: FrescoObject

A range within the document between and including two positions.
boolean equal(in Range rng);
boolean not_equal(in Range rng);
boolean is_before(in Range rng);
boolean is_after(in Range rng);
equal and not_equal are a comparison to another document range and determine if two ranges represent the same piece of a document or not. is_before determines if the given range is before this range or not. is_after determines if the given range is after this range or not.
readonly attribute Position first;
readonly attribute Position last;
begin and end points of the document range.
boolean is_empty();
boolean contains_range(in Range rng);
boolean contains_position(in Position pos);
boolean is_disjoint_from(in Range rng);
containment functions for the range. is_empty determines anything or is simply a position. contains_range determines if the endpoints of the given range are equal to the endpoints of this range or contained within the endpoints of the range. contains_position determines if the given position is equal to one of the endpoints of this range or between them. is_disjoint_from determines if the given range is completely outside of the this range.
boolean intersects(in Range rng);
Range get_intersection(in Range rng);
intersects determines if two ranges intersect and returns true if they do, false otherwise. get_intersection returns the intersection with the given range, or nil if the two ranges don't intersect.
long length();
number of items spaned by the range.
