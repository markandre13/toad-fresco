Button [index]

declared in file: /opt/Fresco/Fresco/defs/widgets.idl
ancestors: Viewer

A button is a viewer that can perform an action when clicked (pressed and subsequently released with the pointer over the button's allocated area). Buttons manipulate a telltale that represents the current state of the button and determines the button's appearance.
Action click_action;
The button will execute its action (if not nil) when clicked.
Glyph label;
The label function defines a Button's appearance
