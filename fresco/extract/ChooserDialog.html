ChooserDialog [index]

declared in file: /opt/Fresco/Fresco/defs/dialogs.idl
ancestors: MessageDialog

A ChooserDialog allows the user to pick an item from a list of choices.
string caption;
attribute string subcaption;
The caption and subcaption strings are used to decorate the chooser viewer.
CosNaming::Name path;
The path is the chosen name.
void choose();
For non-modal dialogs, sets path attribute to currently selected item.
