PopupMenu [index]

declared in file: /opt/Fresco/Fresco/defs/menu.idl
ancestors: Menu

The popup menu is a menu that appears in its own popup window.
void region_alignment(in Alignment x, in Alignment y);
The region alignment determines where the reference point for the popup window is in the region given to show a popup menu.
void menu_alignment(in Alignment x, in Alignment y);
The window alignment determines the alignment of the popup window with respect to the region alignment. If an align menu item is set the menu alignment is relative to that menu item. If it is not set then the menu alignment is relative to the popup window.
