TextFlow::InsertPosition [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl

A place within a document view that represents where things can be inserted into the document. It gives a way to navigate through the document view.
boolean insert_enable;
attribute Position dot;
attribute Position mark;
attribute Element insert_element;
insert_enable controls the visibility of the insert marker. dot is the location within the document subject where edit focus is currently positioned. mark can be used to indicate a selection.
boolean position_from_pick(in GlyphTraversal t, in Vertex pt);
set the insert position from a traversal. The nearest possible location that will allow an insert position is chosen along the hit line. The return value indicates if a position was found.
long forward(in long amount);
long backward(in long amount);
long up(in long amount);
long down(in long amount);
navigation functions. The forward and backward operations will move the insert position forward and backward respectively by the given amount based on mapped view locations that can be visited by the insert marker, or as much as possible if the request can't be fully carried out. The up and down operations move along the minor axis (typically vertical) while attempting to keep the same distance along the major formatted axis (typically horizontal). The amount actually moved is returned.
void end_of_line();
void begin_of_line();
move the insert position of the end/begining of the line it's currently positioned over.
