declared in file: /opt/Fresco/Fresco/defs/dialogs.idl

A MessageDialog is a dialog that has a limited set of user choices.
void message(in Glyph g);
Interactively changes the message of a dialog box
enum Flavor {
  error, information, question, warning, working
};
The message dialog flavors represent the different types of messages the user can see. Show the user the indicated message flavor in the dialog.
enum Choice {
  ok, yes, retry, apply, reset, close, pause, no, stop, resume, cancel,help, none
};
The message dialog choices represent the different choices that the user can make.
void show_choice(in Choice c, in Glyph g, in Action a);
Show the user the indicated choice and invoke the action when the user responds with that choice.
void hide_choice(in Choice c);
Remove the choice.
void choice_action(in Choice c, in Action a);
Change the action for the given choice.
void set_choice(in Choice c);
This operation informs a modal dialog that it should return after processing of the current event is completed and return the choice.
void set_default_choice();
This operation informs a modal dialog that it should return after processing of the current event is completed and return the default choice for the dialog.
Choice get_choice();
Return the users choice.
Action get_action(in Choice c);
Return the action for the given choice.
Choice post_modal_choice(in Window show_for);
This operation posts the dialog in a modal fashion, waits for the user to respond, then returns the user's choice.
Glyph contents(in Viewer container);
Return a glyph to be inside the given containing viewer, if one is desired. If the dialog simply contains a message with no other contents, then this operation returns nil.
