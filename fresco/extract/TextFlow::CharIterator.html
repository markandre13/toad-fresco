TextFlow::CharIterator [index]

declared in file: /opt/Fresco/Fresco/defs/text.idl
ancestors: DocIterator

Allows navigation over the character data of a document.
void next_n(in long n);
void prev_n(in long n);
next_n moves forward n characters, and invalidates the iterator if there are not enough characters to fill the request. prev_n moves backward n characters.
CharCode character();
current character positioned over.
Position before;
attribute Position after;
before is the document position just before the character, and after is the position just after the current character. Setting either of these has the effect of warping the iterator to a new position.
