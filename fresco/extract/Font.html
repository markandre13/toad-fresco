Font [index]

declared in file: /opt/Fresco/Fresco/defs/drawing.idl
ancestors: FrescoObject

A font defines a mapping between character codes and their appearance on the screen.
struct Info {
  Coord width, height;
  Coord left_bearing, right_bearing;
  Coord ascent, descent;
  Coord font_ascent, font_descent;
  
};
The Info structure defines font metric information for a character, string, or summary information for the entire font. The width and height fields define the overall dimensions. The left_bearing and right_bearing fields denote the left and right edges inside the width. The ascent and descent fields define the distance to the baseline from the top and bottom, respectively. The font_ascent and font_descent fields return the summary baseline information for the font, as opposed to the character-specific distances provided by ascent and descent.
boolean equal(in Font f);
unsigned long hash();
The equal operation tests whether two font objects refer to the same font information. The hash operation will return the same value for two equal fonts, though there is also some chance it will return the same value for two unequal fonts.
CharString fullname();
Return the full name of the font. The syntax of the name is platform-dependent.
CharString encoding();
Return the character set identification, such as "iso8859" for ISO Latin.
CharString font_family();
Return the family name of the font. The syntax of the name is platform-independent, i.e., "courier" regardless of where the font came from (Mac, Win32, or X11).
CharString font_style();
Return the style of the font. The syntax of the style is a set of attribute names separated by blank characters, i.e., "", "bold," "bold italic," or "outline condense."
Coord point_size();
Return the point size of the font.
void font_info(out Font::Info i);
void char_info(in CharCode c, out Font::Info i);
void string_info(in CharString s, out Font::Info i);
Return font metric information. The font_info operation returns summary metrics for all characters, char_info returns metrics for a particular character, and string_info returns metrics for a string of characters.
Raster bitmap(in CharCode c);
The bitmap operation creates a bitmap for a given character in the font. In this case the bitmap width and height will reflect the actual size of the character and the bitmap origin will correspond to the character's left bearing and baseline.
