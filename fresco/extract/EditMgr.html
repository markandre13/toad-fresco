EditMgr [index]

declared in file: /opt/Fresco/Fresco/defs/applic.idl
ancestors: FrescoObject

An EditMgr maintains a list of commands for undo/redo. An EditMgr does not normally execute commands. Rather, commands are executed externally and then logged with the EditMgr.
void log(in Command cmd);
void unlog();
The log operation appends a command to the end of the command list. The unlog operation removes the command from the end of the list.
boolean forward();
boolean backward();
These operations move the current command along the list. The forward operation moves it away from the start; the backward moves it toward the start. Both operations return true if successful in moving the current command.
Command current_command();
Returns the current command in the list.
void empty();
void trim();
The empty operation removes all commands from the list. The trim operation truncates the list from one past the current end.
